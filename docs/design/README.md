# 디자인 명세

이 폴더는 서비스의 디자인 명세를 HTML/CSS/JS로 관리합니다.
빌드 없이 브라우저에서 바로 확인할 수 있습니다.

## 사용법

1. HTML 파일을 브라우저에서 직접 열기
2. 상단 케이스 탭으로 상태별 화면 전환
3. 버튼/모달 등 인터랙션 직접 확인

## 구조

```
docs/design/
├── shared/              # 디자인 시스템
│   ├── tokens.css       # 디자인 토큰 (컬러, 타이포, 간격)
│   ├── components.css   # 공통 컴포넌트 스타일
│   └── base.css         # 리셋, 레이아웃 기본
├── web/                 # 채팅 UI 화면별 HTML + .cases.md
└── assets/              # 아이콘, 이미지
```
